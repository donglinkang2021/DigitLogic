    • 半加器：2输入按位加功能模块

    • 输入：X，Y

    • 输出：和位S，进位C

    真值表 表达式 实现电路 卡诺图

    • 全加器：3输入按位加功能模块

    • 输入：X，Y，进位Z

    • 输出：和位S，进位C

    • S 是三位异或函数（奇函数） \(S = X \oplus Y \oplus Z\)

    • C是 \(XY + (X \oplus Y) Z\)

      • 项 \(XY\) 是==进位生成==

      • 项 \(X \oplus Y\) 是==进位传播==

    • 溢出

    ➢ 进位\(C_n\)

    ➢\(V = C_n \oplus C_{n-1}\)

    • 无符号数 unsigned

    • 加法

      • \(C_n\)为1发生溢出

      • \(C_n\)为0无溢出

    • 减法

      • \(C_n\)为1：结果为正，无需校正

      • \(C_n\)为0：结果为负数补码，需要校正

    • 有符号数signed（符号-补码）

    • V=0，无溢出

    • V=1，有溢出，表明运算结果有n+1位，溢出位为符号位
